Installation
============

.. code:: shell

    pip install django-develop

Supported and tested on:

* Python: 2.7, 3.4, 3.5, PyPy
* Django: 1.8, 1.9


Usage
=====

.. code:: shell

    $ django-develop-config
    ...
    $ django-develop <subcommand>
